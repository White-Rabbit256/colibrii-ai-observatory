Sección 01CAPI-CR — Algoritmo de Índice Compuesto

El Colibrii AI Preparedness Index for Costa Rica (CAPI-CR) extiende el AI Preparedness Index (AIPI) del FMI de 4 a 6 dimensiones, añadiendo Energía Sostenible y Seguridad Digital.

Paso 1: Normalización Min-Max

norm(x) = (x − min) / (max − min)
Donde min y max se calculan sobre los 20 países del peer set. Si max = min, todos obtienen 0.5. 
const mm = v => {
  const n = v.filter(x => x != null && !isNaN(x));
  if (!n.length) return v.map(() => null);
  const lo = Math.min(...n), hi = Math.max(...n);
  return hi === lo
    ? v.map(x => x == null ? null : 0.5)
    : v.map(x => x == null ? null : (x - lo) / (hi - lo));
};

Paso 2: Inversión de Indicadores Negativos

El desempleo (SL.UEM.TOTL.ZS) tiene inv: true: valores más altos = peor → se invierte: 1 - norm(x)

Paso 3: Promedio por Dimensión

D_k = avg(indicadores normalizados de la dimensión k)

Paso 4: Override con Puntuaciones Curadas (D4 y D6)

Las dimensiones D4 (Regulación) y D6 (Seguridad) usan puntuaciones curadas manualmente (objeto CUR) porque no tienen datos automáticos del Banco Mundial.

Paso 5: Composite Score

CAPI-CR = Σ(D_k × W_k) para k = 1..6

Sección 21Sistema de Caché (SessionStorage)

// TTL: 30 minutos
function cacheGet(k) {
  const c = sessionStorage.getItem(`clb_${k}`);
  if (!c) return null;
  const d = JSON.parse(c);
  if (Date.now() - d.ts > 30 * 60 * 1000) return null;
  return d.data;
}

function cacheSet(k, v) {
  sessionStorage.setItem(`clb_${k}`, JSON.stringify({
    ts: Date.now(), data: v
  }));
}
